¿Qué es una página local? Indica un ejemplo.
Es un servidor instalado principalmente para trabajar tanto de manera offline como online. Por ejemplo la pagina "Home".

¿Qué es una página remota? Indica un ejemplo.
Significa establecer una configuración de modo que Dreamweaver sea capaz de comunicarse directamente con un servidor en Internet. Como por ejemplo "AnyDesk".

¿Qué se indica en HREF?
Indica el link del sitio destinado.
